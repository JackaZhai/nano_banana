快速掌握平台使用

本手册覆盖仪表盘、API 密钥管理、图像生成和系统设置等常用功能。

快速开始 仪表盘 API 密钥管理 图像生成 系统设置 常见问题

快速开始

1) 启动服务：python app.py
2) 浏览器访问：http://<你的公网IP>:<端口>

仪表盘

积分余额：读取当前 API Key 的余额。
API Key：展示是否已配置有效 Key。
API 主机：显示国内直连或海外节点。
模型状态：显示各模型当前状态，可点击刷新。

API 密钥管理

在“API 密钥”页面输入 Key 并保存。
可测试、删除或切换当前 Key。

图像生成

填写提示词并选择模型、比例、分辨率。
可上传参考图（最多 3 张，每张不超过 5MB）。
点击“生成图像”提交任务，进度会在右侧展示。

系统设置

选择 API 主机：国内直连或海外节点。
设置默认绘图模型。

常见问题

余额显示 --：请确认已配置 API Key。
模型状态异常：查看提示信息后再重试。
